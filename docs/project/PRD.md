# FirinEmek — Ürün Gereksinim Belgesi (PRD)

## Özet

FirinEmek, Türkiye'deki bağımsız artizan fırınlar ve pastaneler için hammadde envanteri, tarif yönetimi, günlük üretim partileri ve toptan sipariş takibi sunan B2B SaaS platformudur.

## Hedef Kitle

- Bağımsız artizan fırınlar
- Butik pastaneler ve viennoiserie atölyeleri
- Toptan ekmek/pastane ürünü satan üreticiler

## Temel Problem

Fırınlar hammadde stoklarını Excel'de, tarifleri kağıtta, üretim programını WhatsApp'ta, toptan siparişleri dağınık defterlerde takip ediyor.

## Değer Önerisi

Tek platformda: hammadde stoku → tarif → üretim partisi → toptan sipariş.

## Tasarım Yönü

**Brutalist Editorial / Artisan Bakery** — Playfair Display + DM Sans, charcoal #1A1A1A + wheat #E8D5B7 + terracotta #C45C3E, sidebar navigasyon.
